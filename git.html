git

  • ls - расшифровывается как "List files in the directory", что переводится как "перечислить все файлы в папке" или "вывести на экран список файлов в папке".
  • cd - расшифровывается как "Change directory", что переводится как "сменить папку" или "перейти в другую папку".
  • pwd - расшифровывается как "Present working directory", что переводится как "текущая рабочая директория".
  • mkdir - расшифровывается как "Make directory", что переводится как "создать папку".
  • mv - расшифровывается как "Move", что переводится как "переместить" или "переименовать".
  • git clone - команда для копирования репозитория
  • git status - отображает состояние рабочей области и индекса
  • git add - добавляет файл(ы) в индекс
  • git commit - сохраняет изменения
  • git push - отправляет изменения в удалённый репозиторий
  • git pull - загружает и сливает изменения с удалённого репозитория
  • git branch - показывает ветки или создает новую ветку
  • git checkout - переключает ветки или восстанавливает файлы
  • git merge - сливает ветки
  • git remote - управляет удалёнными репозиториями
